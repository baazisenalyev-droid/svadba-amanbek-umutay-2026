# 💍 Приглашение на свадьбу Аманбека и Умутай

Элегантный 3D-сайт-приглашение на свадьбу.

## Детали
- **Жених:** Аманбек
- **Невеста:** Умутай
- **Дата:** 26 июля 2026
- **Время:** 16:00
- **Место:** Ресторан «Баласагын», Шамшинская 25а, Токмок, Кыргызстан

## Особенности сайта
- Полноценная **3D-сцена** с вращающимися обручальными кольцами (Three.js)
- **Таймер обратного отсчёта** до начала свадьбы (в реальном времени)
- **Интерактивная карта** (Leaflet + OpenStreetMap) с маркером ресторана
- Кнопка открытия точного местоположения в Google Maps
- Полностью адаптивный дизайн (отлично выглядит на телефонах и планшетах)
- Красивый романтический стиль с золотыми акцентами

## Как запустить локально
Просто откройте `index.html` в любом современном браузере.

## Развёртывание на GitHub Pages (рекомендуется)
1. Перейдите в настройки репозитория: **Settings → Pages**
2. В разделе **Source** выберите:
   - **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`
3. Нажмите **Save**
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://baazisenalyev-droid.github.io/svadba-amanbek-umutay-2026/`

## Технологии
- Tailwind CSS (Play CDN)
- Three.js (3D кольца)
- Leaflet.js (карта)
- Чистый современный HTML + CSS + JS
